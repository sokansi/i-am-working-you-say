# 働いています、とあなたは言う

AIエージェントが「お役所仕草」でタスクを処理する様子を再現するデモページです。
（長期的な運用を意図して作成されていません）

**[デモを見る](https://sokansi.github.io/i-am-working-you-say/)** (GitHub Pages)


## 使い方

GitHub Pages のリンクからブラウザで閲覧できます。

ローカルで実行する場合は `index.html` をブラウザで開いてください。サーバーやビルドツールは不要です。

シナリオを選択して再生ボタンを押すと、エージェントの対話が再現されます。


## 補足

 [Session Spawn](https://github.com/sokansi/session-spawn) の**お役所モード**（カスタムシステムプロンプト）で実際に動作させた際のログを元に構築しています。
